\chapter{Objetivos}
\label{cha:objetivos}

En esta sección del proyecto se detallan los objetivos generales y específicos que se pretenden alcanzar con el desarrollo del mismo, además de establecer un marco para la evaluación continua y la mejora de la experiencia del usuario.

\section{Objetivo General}
\label{sec:objetivo_general}

Identificar sugerencias dentro de comentarios de TripAdvisor para que luego estas puedan ser utilizadas para mejorar la experiencia del turista en plataformas digitales.

\section{Objetivos Específicos}
\label{sec:objetivos_especificos}

\begin{itemize}
    \item Limpieza de los datos obtenidos de las reseñas de usuarios en plataformas digitales de turismo.
    \item Seleccionar un modelo de NLU adecuado
     para analizar las reseñas de usuarios en plataformas digitales de turismo.
    \item Validar la precisión y efectividad del modelo seleccionado en la identificación de sugerencias de los usuarios.
\end{itemize}
