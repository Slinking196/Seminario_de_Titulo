\chapter{Objetivos}

En esta sección del proyecto se detallan los objetivos generales y específicos que se pretenden alcanzar con el desarrollo del mismo, además de establecer un marco para la evaluación continua y la mejora de la experiencia del usuario.

\section{Objetivo General}

Nuestro objetivo general es mejorar las recomendaciones de viajes, hospedaje y servicios relacionados al turismo, gracias a un análisis exhaustivo de las reseñas de usuarios en plataformas como Booking, TripAdvisor y Trivago utilizando técnicas de procesamiento de lenguaje natural (NLP).

\section{Objetivos Específicos}

Nuestros objetivos específicos explicarán cómo se logrará el objetivo general y son los siguientes:

\begin{itemize}
    \item Recolectar reseñas de usuarios de plataformas turísticas digitales tales como Booking, TripAdvisor y/o Trivago utilizando técnicas de web scraping de forma automatizada.
    \item Procesar las reseñas recolectadas utilizando técnicas de procesamiento de lenguaje natural (NLP) para extraer información relevante sobre la experiencia del usuario.
    \item Hacer uso de técnicas de clustering para agrupar las reseñas en categorías temáticas relacionadas a las heurísticas de la TX.
    \item Analizar los datos procesados para identificar patrones y tendencias en las opiniones de los usuarios sobre destinos turísticos, alojamientos y servicios relacionados.
    \item Desarrollar un modelo de clasificación de sentimientos para categorizar las reseñas en positivas, negativas y neutrales.
    \item Analizar a través de herramientas estadísticas y de visualización de datos los resultados obtenidos del análisis de las reseñas, encuestas y entrevistas.
    \item Desarrollar un sistema de recomendaciones personalizadas para mejorar la experiencia del usuario en la planificación de viajes.
    \item Proponer recomendaciones o mejoras para un posible diseño o rediseño de plataformas/aplicaciones que mejoren la experiencia del turista.
    \item Evaluar la efectividad de las recomendaciones implementadas a través de métricas de satisfacción del usuario y análisis de datos.
    \item Realizar un seguimiento continuo de las opiniones de los usuarios para identificar áreas de mejora y adaptar las recomendaciones a las necesidades cambiantes de los turistas.
\end{itemize}
